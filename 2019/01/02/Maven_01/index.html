Maven简介

  • 什么是maven
    maven是apache下的一个开源项目，是纯java开发的，并且只是用来管理java项目
  • maven的好处
    1. 依赖管理：就是对jar包的管理，可以节省空间
    2. 项目一键构建：一个命令tomcat:run就可以把项目运行起来
      编码 编译 测试（junit） 打包 部署
      maven能干的事：编译 测试（junit） 打包 部署
    3. 可以跨平台
    4. 应用于大型项，可以提高开发效率
  • maven的分模块开发
    用户管理 订单管理 支付管理
    互联网项目:按业务分层
    传统项目：按层分 entity（实体） dao sercive web

入门程序

pom.xml：maven项目的核心文件
项目的目录结构：

  • Maven常用命令，执行时前面要加mvn

    • clear：清理已编译好的文件
    • Compile：编译主目录的文件
    • Test：编译并运行测试（Test）目录的代码
    • package：打包
    • install：把项目放到本地仓库
    • Tomcat:run：一键部署

  • Maven的生命周期

    compile test package insatll deploy（发布到私服）
    1. Clean生命周期：Clean
    2. Default生命周期：compile test package insatll
    3. Site生命周期：Site——生成一个site文件夹，生成项目的站点文档

  • 不同的生命周期的命令可以同时执行
